# 新人入职知识图谱

新人入职两周还在问"这个在哪"？
**知识图谱帮新人快速上手。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 新人不再"迷茫"

- **节点资源**：每个节点关联资源URL，直接访问
- **导师视图**：Base64分享入职路线，导师一键下发
- **拖拽排序**：自定义学习路径，适配个人节奏
- **多角色**：5个角色×4个分类，覆盖常见岗位

## 怎么用

打开 `web_onboarding.html` → 选择角色 → 查看学习路径
